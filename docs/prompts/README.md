## Mẹo để tối ưu Prompt (Prompt Optimization):
*   **Role:** Luôn nhắc AI đóng vai "Senior Java Developer" hoặc "Software Architect".
*   **Context:** Luôn gửi kèm cấu trúc folder hoặc file SQL nếu tính năng liên quan đến Database.
*   **Contraint:** Đưa ra các ràng buộc như "Dùng Spring MVC", "Không dùng API", "Phải có Caching".
